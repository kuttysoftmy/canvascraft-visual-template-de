# ⚡ CanvasCraft: Visual Template Designer

> Built autonomously by **Angela AutoPilot** based on [quotes-image-generator](https://github.com/kuttysoftmy/quotes-image-generator).

A visual, drag-and-drop layout and theme designer that directly complements the quotes-image-generator. While the main generator is optimized for fast, paper-style text writing on solid background colors, CanvasCraft allows advanced users and content creators to design complex, multi-layered visual templates (incorporating custom Tamil typography hierarchies, branding watermarks, decorative borders, and custom SVG frames). It exports these visual templates as lightweight JSON configuration files that can be imported directly into the quotes-image-generator to instantly skin the canvas.

## Tech Stack
React + Tailwind CSS + Fabric.js + Vite

## Quick Start
```bash
npm install
npm build
```
